New Inovations
 The R&D Team as developed a security platform that enhances and provides additional security for WEB based applicaitons. In response to a customer requirements. The EQUILIZER, (not misspelled) is a firewall implementation that requires individuals to be authenticated prior to being passed to the Web based application. The Equilizer can and does provide additional information and logs access information of the users. The security gateway also limits access by source address based on user authentication criteria.
