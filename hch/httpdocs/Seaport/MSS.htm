Managed Security Solutions

HCH Enterprises Managed Security Solution (MSS) is a full suite of integrated security products and services that are assembled to provide your business with a complete security offering. Your mission-critical systems are integrated with monitoring tools and utilities that are widely used in the industry along with our best in practice methodologies. Public Key Infrastructure (PKI) can be implemented along with virtual private networking (VPN), e-commerce based applications using secure sockets layer (SSL), and other applications requiring scalable enterprise security management.


With MSS E-mail Security is implemented with Anti-Span solutions, Filters, and Junk E-mail blockers. Windows Security services utilizing 2000 Active Directory (AD) and complete user profile and roaming profiles, user quotas and advance file server resource design and implementations.


Anti-Virus services are established within your systems and network infrastructure using server based enterprise solutions for Windows 2000, and Linux environments. Solutions are implemented to enhance Unix security with LDAP, TACCACs and TACCACs +, or Radius. System evaluation and lock down of un-used or vulnerable Unix services.


Auditing and Spying Tools are used as a management control to facilitate the organizational goals and security policies of your company. PKI can be implemented with various products provided by companies such as Cisco, Checkpoint, and Microsoft. Whenever feasible, we will use your existing appliances within the proposed architecture. In addition, forensic intrusion detection systems (FIDS) can be implemented, using Linux-based open source applications, as well as other commercially available solutions, which continually monitor authorized and non-authorized activity to track, and maintain documented evidence of security violations should they occur.
